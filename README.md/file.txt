# AI-Powered Self-Healing Cloud Security System

This project is designed to detect cyber threats using Artificial Intelligence and automatically protect cloud infrastructure.

## Project Structure

- frontend → user interface
- backend → server and API
- ml-model → AI threat detection model
- docs → project documentation

## Goal

Build a system that detects suspicious activity and automatically protects the cloud environment.
